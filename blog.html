1. What is DOM and what is the purpose of DOM?

In JavaScript, DOM is actually a class that makes it really easy to define each element present within a web page so it can be modified later. Whenever a page is loaded in the browser, its document object is formed that is named as "document". The Document Object Model (DOM) aims to make it easier for Javascript to access HTML elements. DOM plays a rather very important role in website performance. For example, let's say you are visiting a website and logging into your account. After logging in, you will be greeted by the website, for example, 'Hello ' along with your username, profile picture, etc.

3. What is event bubble?

Event Bubbling in Javascript describe in which order some nested elements receive a given event.The principle of bubbling is simple. Whenever an event occurs on an element, at first place it will run the handler on it, then its parent, then on other ancestors.

4. What is a callback function and why will you use it?

A JavaScript callback is a function which is to be executed after another function has finished execution. A more formal definition would be - Any function that is passed as an argument to another function so that it can be executed in that other function is called as a callback function. In our event handler section we put the callback function so it can do the work inside and return the value to the following element where it will work upon.
